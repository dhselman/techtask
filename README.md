# TechTask MongoDB Viewer

A single Node.js application that connects directly to MongoDB at 20.42.40.33 and displays all databases, collections, and documents in a simple web UI.

## Features
- Connects directly to MongoDB (no backend API layer)
- Lists all databases, collections, and documents
- Simple web interface

## Usage
1. Install dependencies: `npm install`
2. Start the app: `node app.js`
3. Open your browser at http://localhost:3000

## Requirements
- Node.js 18+
- Network access to MongoDB at 20.42.40.33:27017
